# Sistema de Asignación de Citas - Salón de Belleza

¡Bienvenido! Este es un proyecto de práctica para desarrollar el pensamiento lógico, estructura y arquitectura de software desde cero.

## 🛠️ Tecnologías utilizadas hasta ahora
- **Backend:** Node.js con Express.
- **Formato de datos:** JSON.

## 📋 Reglas de Negocio actuales
1. El salón cuenta con dos estilistas independientes: **Jenny** (id: `est_01`) y **Marcela** (id: `est_02`).
2. El servidor expone los datos de las estilistas en la ruta `/estilistas`.
3. Se pueden simular reservas enviando un método `POST` a la ruta `/citas`.
